## Tema básico para Blogs en WordPress
**Tema híbrido a caballo entre el método cásico y el método de bloques**. Desarrollado para un blog cualquiera, diseñado para ofrecer un blog rápido, moderno y completamente administrable desde el panel de WordPress. Basado en Stories V2, hecho para ser fácilmente extensible y usarse como base para otros proyectos

## Requisitos
- WordPress 6.0+
- PHP 8.0+
